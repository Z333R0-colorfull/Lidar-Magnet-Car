
# Lidar-Magnet-Car

> 基于 insani-bot 的 Gcode 方案  
> **超低成本激光雷达小车**  
> 实现全向运动、雷达解算、低速建图、磁传感器读取与滤波

---

---

## ⚠️ 重要提示

- **必须调平重心**，否则运动极不稳定  
- 推荐加装编码器 + 陀螺仪 → 显著降低漂移  
- 用热缩管套住滚子可提高抓地力
